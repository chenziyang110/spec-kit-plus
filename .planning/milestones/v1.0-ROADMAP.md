## Milestone Summary

**Key Decisions:**

- Use `pydantic-graph` for the investigation lifecycle.
- Use Markdown for investigation state persistence and resume.
- Enforce "No Fix Without Proof" before autonomous fixing proceeds.
- Escalate to `AwaitingHumanNode` after repeated verification failures.

**Issues Resolved:**

- Persistent investigation graph and session resume support.
- Artifact-aware context loading and recent-change prioritization.
- Verification-led fixing with persisted human handoff reporting.

**Issues Deferred:**

- Automated knowledge base for resolved bugs.
- Multi-model routing for cheaper gather/summarize versus expensive investigation.
- Deep tracing and observability instrumentation.

**Technical Debt Incurred:**

- Milestone archival commit and git tag were deferred in the dirty workspace and should be handled once the worktree is stabilized.
